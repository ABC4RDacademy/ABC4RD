# Article Preview Assets

This folder stores thematic preview images for public ABC4RD Academy articles.

## Rule

Every public article, bulletin, essay, digest, or major publication should have
one small thematic preview image stored here or in a clearly related asset path.

## Naming

Prefer stable descriptive names:

- `2026-05-12-global-public-signal-top-20.svg`
- `research-bulletin-002.svg`
- `trustworthy-computation-essay.svg`

## Format

Preferred first format:

- SVG for diagram-like or poster-like covers

Use PNG when:

- the image depends on raster art;
- screenshots are required;
- gradient and texture quality matters more than vector simplicity.

## Language

One visual concept may be reused across the six UN-language versions of the
same article. Translate the alt text in the document itself.

## Safety

Do not place private, licensed, or endorsement-implying artwork here.
