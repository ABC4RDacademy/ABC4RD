# Website Builder Open-Source Watchlist

ABC4RD Academy is watching these repositories to study open-source approaches
to building landing pages, documentation hubs, CMS-backed sites, visual editors,
commerce sites, and reusable UI systems.

This is not a partnership list.

|---|---|---|

| https://github.com/webstudio-is/webstudio | Open-source Webflow-style website builder | starred/watching |
| https://github.com/silexlabs/Silex | Visual static site builder | starred/watching |
| https://github.com/puckeditor/puck | Visual editor for React | starred/watching |
