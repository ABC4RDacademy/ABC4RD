# ABC4RD Crypto Donations Policy

Status: v0.1 draft.

ABC4RD Academy may accept crypto donations only when the legal/accounting and wallet security process is ready.

## First Rule

Do not use crypto donations to promote speculation, tokens, yield, investment returns, or trading.

## Initial Scope

Recommended first crypto rail:

- Bitcoin donations;
- Lightning payments;
- BTCPay Server.

Do not start with:

- ABC4RD token sale;
- meme coins;
- illiquid tokens;
- anonymous large token donations;
- DAO treasury fundraising;
- stablecoin custody without accounting/legal review.

## Why BTCPay Server

BTCPay Server is open-source and self-hosted. It fits the Academy's Bitcoin and open infrastructure positioning.

Tradeoff:

- more sovereignty;
- more operational responsibility.

## Donation Handling

For each crypto donation record:

- date/time;
- asset;
- amount;
- transaction ID if public;
- fiat value at receipt time;
- donor note if provided;
- campaign;
- conversion decision;
- custody location;
- accounting status.

## Wallet Security

Minimum rules:

- no private keys in GitHub;
- no seed phrases in cloud notes;
- hardware wallet or secure wallet for treasury;
- limited hot wallet balance;
- backups stored offline;
- large transfer approval by at least two trusted people;
- public donation address verified through official Academy channels.

## Refunds

Crypto refunds are manual.

Refunds should consider:

- original transaction amount;
- fiat value at payment time;
- network fees;
- volatility;
- sanctions/compliance risk;
- original sender verification.

## Compliance Warning

Large, suspicious, sanctioned, or restricted donations must be reviewed before acceptance/use.

ABC4RD Academy should not promise anonymity to donors.

## Public Disclaimer

Suggested wording:

`Crypto donations support ABC4RD Academy educational work. They are not investments, token purchases, membership securities, or claims on future revenue. ABC4RD Academy does not provide financial, investment, legal, tax, or trading advice.`
