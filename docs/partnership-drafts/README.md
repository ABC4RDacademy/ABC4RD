# Partnership Drafts

This folder is for draft notes only.

Nothing in this folder means that ABC4RD Academy has a partnership,
sponsorship, endorsement, or official relationship with any organization.

Allowed:

- preferred official links;
- draft source correction requests;
- draft guest-session invitations;
- draft event notes;
- public source summaries.

Not allowed:

- private emails;
- private phone numbers;
- secrets;
- contracts;
- internal negotiation notes;
- claims of partnership without explicit public approval.
