# ABC4RD Nanomaterials Lab Open-Source Stack Map

ABC4RD Nanomaterials Lab is the Academy block for materials informatics, nanomaterials datasets, simulation workflows, computational materials science, AI-assisted discovery, reproducibility, and responsible scientific infrastructure.

## First Scope

| Layer | Open-source stack | First pilot | Time | Cost |
|---|---|---|---:|---:|
| Materials analysis | pymatgen, Materials Project API | crystal structure notebook | 1-3 days | $0 |
| Materials workflows | atomate2, custodian, FireWorks | reproducible workflow study | 3-14 days | $0-150/month |
| Data mining | matminer, matgenb | materials feature notebook | 1-5 days | $0 |
| Research archive | NOMAD, AiiDA | provenance and metadata study | 1-3 weeks | $50-300/month |
| Simulation | LAMMPS, DeepMD-kit | molecular dynamics concept lab | 1-4 weeks | $0-500/month compute |
| AI discovery | DeepMind materials discovery | literature and dataset review | 2-10 days | $0 |

## First Academy Product

Create a nanomaterials notebook lab: load a public material structure, explain properties, cite sources, and show why provenance matters.

Production note: no claims about material safety, clinical use, manufacturing performance, or commercialization without expert review and experimental validation.
