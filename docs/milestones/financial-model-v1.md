# Financial Model v1

Status: active operating milestone.

Goal: define the first ABC4RD Academy revenue model without expanding into
uncontrolled offers, token promises, or territory-specific legal complexity.

## Scope

- decide what ABC4RD sells;
- decide first price ranges;
- define first buyer segments;
- package by language group before territory;
- define fiat, crypto, grant, sponsorship, in-kind, and service-exchange value;
- define where money goes after receipt;
- create GitHub issues for the first operating approvals.

Use fiat first. Keep crypto as a reviewed pilot. Start with one course, one
certificate review batch, and one institutional starter offer.

- financial model docs are merged;
