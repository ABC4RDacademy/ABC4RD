# Metrics Dashboard

This dashboard defines what ABC4RD Academy should measure every week.

## Public work

| Metric | Current source | Weekly target |
| --- | --- | --- |
| Open work items | GitLab work items | increase with useful, scoped tasks |
| Closed work items | GitLab work items | close at least one meaningful item |
| Merge requests | GitLab merge requests | one reviewed improvement per week |
| Course modules | `LLMAIX2001a` | one module or lab update per week |
| Research citations | `abc4rd-research` | one verified source note per week |
| Translation tasks | all projects | one source-parity translation issue per week |
| Published artifacts | Pages, README, docs | one approved artifact per week |

## Community growth

Track:

- contributors;
- stars and forks;
- opened and closed issues;
- merged merge requests;
- community members;
- approved partner conversations;
- event watch notes;
- citations added;
- translations added.
- ranked entities refreshed;
- weekly ranking baskets published;
- Top 20 movement notes published.

## Review cadence

- Monday: review open work and pick priorities.
- Wednesday: check research and source progress.
- Friday: prepare a public recap draft.
- Monthly: publish a milestone report.
