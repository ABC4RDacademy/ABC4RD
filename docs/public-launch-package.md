# Public Launch Package

Status: draft for maintainer review.

This package is the public launch surface for ABC4RD Academy. It collects the
repository story, site structure, roadmap, digest, partner/event pipeline, and
task routing into one reviewable bundle.

## Launch Message

ABC4RD Academy is an international research and education hub for Bitcoin, AI,
open compute, robotics, sensor networks, digital health, digital manufacturing,
nanomaterials, and source-backed open infrastructure. The public work starts
from visible artifacts: course updates, research sources, event watch notes,
starter work items, and draft-first community recaps.

## Public Claims Boundary

Use:

- ABC4RD has a documented 2017 public media trail through Bitcoin Magazine and
  Nasdaq syndication.
- ABC4RD is building source-backed education and research artifacts.
- ABC4RD is watching relevant ecosystems and events.

Do not use:

- confirmed partnership;
- sponsorship;
- endorsement;
- official event participation;
- medical, financial, or investment advice;
- claims that ABC4RD has built AGI or ASI systems.
