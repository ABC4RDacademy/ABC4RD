# IP And Publication Policy

ABC4RD Academy uses GitHub as a public timestamped work record, but GitHub is
not a substitute for a patent strategy, copyright registration, or legal review.

## Core Distinctions

| Item | What GitHub helps with | What GitHub does not replace |
|---|---|---|
| Copyright | Public record of original text, code, docs, diagrams, and history | Formal registration where required for litigation strategy |
| Patent | Defensive publication and prior-art evidence | Patent filing, prosecution, and international novelty strategy |
| Trade secret | Not suitable if secrecy matters | Confidentiality, NDAs, internal controls |
| Reputation | Proof of work, source discipline, review process | Legal ownership by itself |

## Publication Decision Tree

Before publishing a substantial idea, ask:

1. Is this educational text, documentation, code, or source commentary?
   - Publish with license, authorship, and source notes.
2. Is this a technical invention that may be patentable?
   - Send to IP review before public release.
3. Is this a trade secret or partner-sensitive material?
   - Keep private.
4. Is this third-party material?
   - Link and summarize; do not copy unless license allows redistribution.
5. Is this medical, financial, robotics, nanomaterials, or safety-critical?
   - Add disclaimer and avoid deployment claims.

## Defensive Publication Standard

If the goal is to prevent others from patenting a concept, publish enough detail
to be meaningful prior art:

- problem;
- architecture;
- components;
- workflow;
- diagrams or pseudocode;
- constraints;
- examples;
- author/date/version;
- public release tag.

Vague slogans are weak defensive publications.

## Repository Notices

Use this baseline notice in public research and education repositories:

```text
Copyright (c) 2026 ABC4RD Academy contributors.

This repository contains original educational and research materials.
Third-party sources are cited for reference only and are not redistributed
unless their license permits it.
```

## Patent Caution

Public disclosure can become prior art. In some jurisdictions, publishing
before filing may destroy patentability. If commercial patent protection is
important, file or review before publishing.

## Trade Secret Caution

Do not publish private algorithms, unpublished patentable implementations,
partner-sensitive materials, datasets, credentials, customer data, or internal
strategy if secrecy is part of the value.
