# GitHub Discussions Seed Pack: 2026-05-09

These discussion drafts are designed to create useful public activity without
mass outreach. They should be posted only in ABC4RD Academy repositories where
GitHub Discussions are enabled.

## Posting Rules

- Ask for sources, corrections, and learning paths.
- Do not ask for promotion.
- Do not tag external speakers or maintainers unless they already opted in.
- Keep every discussion tied to a repository file or curriculum module.
- Close the loop by linking accepted sources back into the relevant README.

## Discussion 1: Source Verification

Category: `Source verification`

Title:

```text
Source review queue for ABC4RD Academy research tracks
```

Body:

```text
ABC4RD Academy is building source-backed educational material across Bitcoin,
AI, open compute, sensors, robotics, digital health, manufacturing, and
nanomaterials.

This discussion is for public source suggestions and corrections.

Please include:
- source title;
- official link;
- language;
- license or reuse note if visible;
- track relevance;
- whether it should be marked `verified`, `requires verification`,
  `license unclear`, or `rejected`.

We do not accept pirated PDFs, unclear copies, private materials, or sources
that cannot be publicly verified.
```
