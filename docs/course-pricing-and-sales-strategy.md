# ABC4RD Course Pricing and Sales Strategy

Status: v0.1 draft.

## First Course to Sell

`Blockchain Foundations: From Bitcoin to Verifiable Infrastructure`

## Offer Structure

| Offer | Price idea | What is included |
|---|---:|---|
| Free public course | $0 | GitHub modules, reading list, challenge instructions |
| Certificate review | $10-50 | Submission review and certificate verification page |
| Beta cohort | $49-99 | live sessions, review, certificate, portfolio help |
| Standard cohort | $149-299 | more review, mentor Q&A, project feedback |
| Pro developer cohort | $500-1500 | code labs, testnet demo, office hours |
| Corporate workshop | $2,000-10,000 | private team training |

## Sales Message

ABC4RD Academy teaches blockchain not as speculation, but as verifiable digital infrastructure.

Students learn:

- Bitcoin fundamentals;
- Ethereum and smart contracts;
- tokens and credentials;
- DAO governance;
- oracle risk;
- security basics;
- GitHub contribution workflow;
- portfolio-ready work.

## Scholarship Model

Reserve scholarship seats:

- 20-50% of early cohort seats;
- supported by donations and sponsors;
- selection based on motivation and need;
- no private sensitive data in public GitHub.

## Revenue Projection Example

Conservative monthly beta:

- 20 students at $49 = $980;
- 10 certificate reviews at $25 = $250;
- 20 supporters at $10/month = $200;
- one small sponsor = $1,000.

Total:

- $2,430/month before fees and taxes.

This is not a forecast. It is a planning example.
