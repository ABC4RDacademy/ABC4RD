# Weekly Launch Rhythm

Status: source.

ABC4RD Academy should show steady public progress without expanding
chaotically. The weekly rhythm keeps course work, demo work, research, and
credential review moving together.

## Weekly Cadence

| Day | Public output | Review rule |
| --- | --- | --- |
| Monday | Course or module update | Link to a course file, issue, or pull request |
| Wednesday | Lab or demo update | Include run instructions, test result, or screenshot |
| Friday | Research Bulletin plus ranking update | Use source-backed claims, ranking formula, and approval checklist |
| Monthly | Certificate and review batch | Review evidence before issuing any certificate |

## Monday: Course Or Module Update

Output one concrete learning artifact:

- module outline;
- exercise;
- reading path;
- glossary update;
- teacher note;
- beginner issue.

For Launch Sprint 02, the Monday artifact is:

`docs/course-updates/2026-05-11-course-01-operating-update.md`

## Wednesday: Lab Or Demo Update

Output one runnable or reviewable lab change:

- demo instructions;
- test result;
- local setup note;
- known limitation;
- learner reflection prompt.

For Launch Sprint 02, the Wednesday artifact is:

`docs/demo-updates/2026-05-13-erc20-localnet-review.md`

## Friday: Research Bulletin

Output one public bulletin:

- what changed;
- what was studied;
- what sources or projects are being watched;
- what students can do next;
- what needs maintainer approval.

If a ranking basket is active that week, publish alongside it:

- updated Top 20;
- index formula;
- delta note;
- source table reference.

For Launch Sprint 02, the Friday artifact is:

`docs/bulletins/2026-05-15-research-bulletin-002.md`

## Monthly: Certificate And Review Batch

Review completion evidence in batches. Do not issue certificates
automatically.

Minimum evidence:

- course artifact;
- lab or demo artifact;
- reflection or source note;
- reviewer approval;
- public evidence URL.

For Launch Sprint 02, the monthly certificate artifact is:

`docs/certificates/review-batch-2026-05.md`

## Do Not Do This Week

- Do not add another major track unless it directly supports Launch Sprint 01.
- Do not claim partnership, endorsement, accreditation, investment value, or
  official event participation.
- Do not publish private messages, contacts, credentials, or contracts.
- Do not turn the demo token into a real token launch.
