## Outreach draft template

```text
Hello [name/team],

ABC4RD Academy is building public learning materials around [topic]. We are
preparing [artifact] and would value source-backed feedback from people working
near [event/ecosystem].

The current public work is here: [GitHub issue or document link].

This is not a partnership announcement or sponsorship request. It is a draft
education artifact for review.
```

## Approval checklist

- Target is publicly identifiable.
- Message links to a real public artifact.
- No private credentials, emails, contracts, or CRM data are included.
- Message does not imply partnership, sponsorship, endorsement, or approval.
- Human maintainer approved the exact text.
- Follow-up owner is named.
- Result will be recorded back into GitHub.

## Weekly operating rule

Open one watch issue or draft per week. Do not send mass invitations. Do not
repeat messages across channels. Build a visible trail of useful artifacts first.
