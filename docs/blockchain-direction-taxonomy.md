# Blockchain Direction Taxonomy

ABC4RD Academy should not rank "blockchain" as one flat industry. The sector
must be split into directions first, then ranked direction by direction using
the same index family.

## Operating Rule

For blockchain-sector rankings:

1. assign every entity one primary direction;
2. allow up to two secondary directions;
3. score every entity with the same index set;
4. publish separate Top 20 rankings by direction;
5. optionally publish a larger blockchain family ranking after direction-level
   rankings are stable.

## Index Overlay

Every direction uses the same indices:

- `TSI`
- `OAI`
- `EPI`
- `KAI`
- `CRI`
- `TGI`
- `MI`

This keeps direction-level rankings comparable without pretending that all
business models are identical.

## Ranking Layers

ABC4RD should publish three layers:

1. Direction rankings
2. Blockchain family ranking
3. Cross-sector Top 20

## Direction Ranking Formula

```text
Direction Weekly Score =
TSI + OAI + EPI + KAI + CRI + TGI + MI
```
