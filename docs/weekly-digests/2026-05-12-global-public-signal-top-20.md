## Why These Entities Rose To The Top

This seed basket rewards a simple pattern:

- visible official product surface;
- public technical or documentation footprint;
- real event presence on official pages;
- clear pathways for respectful, non-spam contact.

That is why the first list leans toward organizations with:

- official event programs in 2026;
- official public docs;
- public infrastructure positioning;
- public-facing ecosystem coordination.

## How To Use This Ranking

ABC4RD should use this list to drive:

- weekly research monitoring;
- event and conference review;
- official-source tracking;
- careful outreach draft preparation;
- guest-session invite candidates;
- upstream contribution targeting.

It should not be used to:

- imply endorsement;
- imply partnership;
- mass message companies;
- publish private contact data;
- make investment recommendations.

## Operational Use

For each ranked entity, ABC4RD should keep:

- one official source URL;
- one official event or update URL;
- one public collaboration route, if available;
- one internal next action.

The ranking should be refreshed weekly and published with a short delta note:

- who moved up;
- who moved down;
- which official signal changed;
- which entities were added or removed.
