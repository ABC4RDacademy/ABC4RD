## What Students Can Do Now

Students can start with:

1. read the flagship course outline;
2. complete Day 1 of the challenge;
3. write a short source-backed note;
4. open a GitHub PR;
5. build a first public portfolio artifact.

## What Experts Can Review

Experts can help by reviewing:

- technical accuracy of blockchain modules;
- glossary terms;
- source quality;
- security disclaimers;
- token/RWA boundaries;
- DAO governance examples;
- certificate language.

## What We Are Not Doing

ABC4RD Academy is not:

- selling tokens;
- giving investment advice;
- claiming official partnership without public confirmation;
- issuing accredited degrees;
- deploying production smart contracts;
- using real patient, industrial, or private student data in public demos.
