## Purpose

This dashboard is the first official weekly delta layer for the ABC4RD ranking
system.

It does not fabricate movement where no second published cycle exists yet.

Instead, it publicly confirms:

- which ranking directions are already baseline-locked;
- which directions are not yet delta-ready;
- the exact next comparison date for the first true week-over-week movement
  pass.

## Current Status

All currently published direction rankings dated `2026-05-12` are first-edition
public baselines.

That means:

- all current scores are active and public;
- all current Top 20 tables are official;
- all current baseline rows are already archived;
- but `entered`, `left`, `moved up`, and `moved down` are not published yet,
  because there is not yet a second official dated comparison cycle.

The first true delta-ready cycle will be the Friday publication dated
`2026-05-15`.
