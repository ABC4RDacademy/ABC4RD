# Launch Sprint 01: ABC4RD Academy Public Baseline

Status: draft for maintainer review

Date: 2026-05-11

## Goal

Make ABC4RD Academy understandable, navigable, and useful in one public GitHub
pull request. A visitor should be able to answer three questions in 30 seconds:

- What is ABC4RD Academy?
- Which labs and courses are open first?
- Where can I start or contribute today?

## Scope

This sprint creates the first visible public baseline:

1. A clearer "What is ABC4RD Academy" homepage path.
2. The first blockchain course outline.
3. A test completion certificate template.
4. ABC4RD Research Bulletin #1.
5. A runnable ERC-20 localnet learning demo.

## GitHub milestone

Suggested milestone:

`Launch Sprint 01: ABC4RD Academy public baseline`

## Pull request

Suggested PR title:

`Launch ABC4RD Academy baseline: homepage, first course, certificate, bulletin, demo`

Suggested PR summary:

- Adds the first blockchain course page and learner path.
- Adds the first certificate template and review rules.
- Publishes Research Bulletin #1 as a draft public artifact.
- Adds an ERC-20 localnet demo for supervised learning.
- Updates the homepage with direct start links.

## Definition of done

- Homepage links to the course, certificate, bulletin, and demo.
- Course page has modules, labs, outcomes, and review gates.
- Certificate template is clearly marked as test/draft.
- Bulletin uses only public-safe claims and internal repository links.
- Demo contains no real token launch language, price language, or investment
  language.
- No secrets, private emails, credentials, CRM data, or unpublished contracts
  are included.
