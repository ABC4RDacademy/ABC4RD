# ABC4RD Open Compute Lab Open-Source Stack Map

ABC4RD Open Compute Lab is the Academy block for high-performance computing, distributed compute, AI workloads, scientific workflows, clusters, scheduling, reproducibility, and "unbounded compute" literacy.

## First Scope

| Layer | Open-source stack | First pilot | Time | Cost |
|---|---|---|---:|---:|
| Cluster scheduling | Slurm | job scheduler concept lab | 1-5 days | $0-100/month |
| Containers/orchestration | Kubernetes | student compute cluster demo | 2-10 days | $20-300/month |
| Distributed Python | Ray, Dask | parallel workload demo | same day-3 days | $0-100/month |
| Cloud/IaaS | OpenStack | infrastructure architecture study | 2-8 weeks | $300-3000/month |
| MPI/HPC | Open MPI | parallel computing lesson | 1-5 days | $0 |
| Scientific workflows | Nextflow, Snakemake, Spack | reproducible pipeline demo | 1-10 days | $0-100/month |
| Notebook platform | JupyterHub, Nebari | Academy compute notebooks | 2-14 days | $20-300/month |

## First Academy Product

Create an ABC4RD compute classroom: students run small parallel jobs, compare local vs distributed execution, and learn cost-aware compute planning.

Production note: "unbounded compute" is a concept, not a promise. Real compute has budget, energy, security, quota, and environmental constraints.
