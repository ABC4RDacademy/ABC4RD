# GitHub Discussions

Enable GitHub Discussions for the organization-facing repositories and create these categories.

| Category | Format | Purpose |
| --- | --- | --- |
| Announcements | Announcement | Public updates, releases, milestones |
| Research | Open-ended discussion | Evidence review, source maps, research notes |
| Course Development | Open-ended discussion | Modules, labs, translations, learning outcomes |
| Events | Open-ended discussion | Conference activation, workshops, hackathons |
| Partners | Open-ended discussion | Partner leads, sponsor ideas, ecosystem mapping |
| Community Help | Q&A | Contributor questions and onboarding help |

## Setup notes

GitHub does not fully manage Discussions categories through normal repository files. Treat this document as the source of truth, then create categories manually in repository settings or with approved GitHub API commands after `gh auth login`.
