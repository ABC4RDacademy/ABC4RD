# Start Here: ABC4RD Academy

Status: source.

ABC4RD Academy is an international blockchain and AI education initiative
building open courses, research labs, certificates, and demo projects for
verifiable digital infrastructure.

This page is the simplest public route for new visitors, learners,
contributors, reviewers, and partners.

## 1. Who We Are

ABC4RD Academy teaches technology through open sources, practical work, careful
attribution, responsible experimentation, and public learning infrastructure.

The Academy is independent. Unless explicitly stated, ABC4RD Academy is not
officially affiliated with referenced companies, conferences, universities,
sponsors, speakers, projects, or organizations.

## 2. Languages

English is the canonical GitHub working language.

Public entry points are being prepared in the six official United Nations
languages:

- English;
- Arabic;
- Chinese;
- French;
- Russian;
- Spanish.

Translation status and review rules are defined in
[`docs/i18n/translation-policy.md`](i18n/translation-policy.md).

## 3. First Course

Start with Course 01:

[`Blockchain Academy: From Bitcoin to Verifiable Infrastructure`](courses/blockchain-academy-from-bitcoin-to-verifiable-infrastructure.md)

The course starts with Bitcoin foundations, introduces smart-contract mental
models carefully, and connects the work to verifiable credentials, provenance,
and trust limits.

## 4. First Demo

Run or review the educational localnet demo:

[`ERC-20 Localnet Learning Demo`](../demos/erc20-localnet/README.md)

The demo token has no real-world value, no price, no sale, no fundraising
purpose, and no mainnet deployment plan.

## 5. Test Certificate

Review the first certificate template:

[`ABC4RD Blockchain Foundations Completion Certificate`](certificates/blockchain-foundations-certificate.md)

This is an educational completion record. It is not an accredited degree,
license, employment guarantee, or financial qualification.

## 6. How To Contribute

Start with one small public artifact:

- suggest one glossary term;
- review one course paragraph;
- run the local demo and report the result;
- propose one source for a research track;
- improve one translation while preserving disclaimers;
- open one focused issue from the launch sprint issue list.

Use:

- [`docs/github-launch-issues.md`](github-launch-issues.md);
- [`CONTRIBUTING.md`](../CONTRIBUTING.md);
- GitHub Issues and Discussions.

## 7. Weekly Rhythm

ABC4RD Academy keeps movement visible through a simple operating rhythm:

- Monday: course or module update;
- Wednesday: lab or demo update;
- Friday: research bulletin;
- Monthly: certificate and review batch.

See [`docs/weekly-launch-rhythm.md`](weekly-launch-rhythm.md).
