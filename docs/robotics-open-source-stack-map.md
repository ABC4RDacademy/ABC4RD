# ABC4RD Robotics Lab Open-Source Stack Map

ABC4RD Robotics Lab is the Academy block for robot software, simulation, autonomy, manipulation, fleet coordination, safety boundaries, and student robotics exercises.

## First Scope

| Layer | Open-source stack | First pilot | Time | Cost |
|---|---|---|---:|---:|
| Robotics middleware | ROS 2 | ROS 2 workspace and node graph lesson | 1-2 days | $0 |
| Simulation | Gazebo, Webots, MuJoCo | mobile robot in simulated lab | 1-5 days | $0-50/month |
| Navigation | Nav2 | indoor navigation map demo | 2-7 days | $0-100/month |
| Manipulation | MoveIt 2 | simulated robotic arm pick/place | 3-10 days | $0-150/month |
| Fleet orchestration | Open-RMF | multi-robot workflow study | 1-3 weeks | $50-300/month |
| Autonomous mobility | Autoware | autonomy architecture reading lab | 2-6 weeks | $100-500/month |

## First Academy Product

Create a simulated robotics classroom: students open a browser or local container, see a robot, inspect sensors, move it in a map, and learn why robotics requires simulation before physical deployment.

Production note: no physical robot deployment without safety review, emergency stop procedure, and responsible supervisor.
