## Milestones

### 7 Days

- Publish research tracks, contribution ladder, legal/ethical framework, and
  weekly digest template.
- Create six adjacent curriculum repositories.
- Add at least 50 literature index entries.
- Enable GitHub Discussions where appropriate.

### 30 Days

- Publish first complete module in each new curriculum.
- Complete one external documentation PR or issue.
- Publish four weekly research digests.
- Request accuracy feedback on one narrow source-backed topic.

### 90 Days

- Turn two draft tracks into active tracks.
- Build a reviewed source library with 150+ entries.
- Host one public reading session or expert feedback session.
- Document merged external contributions.
