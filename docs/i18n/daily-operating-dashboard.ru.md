Это публичная ежедневная панель управления для ABC4RD Academy.

Она превращает режим `24/7` в понятный повторяемый daily package без шума.

## Что здесь закреплено

- ежедневные source-блоки;
- ежедневные curriculum-обновления;
- ежедневная дисциплина по issue, draft и digest-слою;
- подготовка weekly ranking до пятничной публикации.
