# ABC4RD Academy

翻译状态：draft, requires native review.

ABC4RD Academy 是一个国际研究与教育项目，关注区块链、Bitcoin、人工智能、开放计算、机器人、传感器、数字健康、数字制造、纳米材料、网络以及可验证的数字基础设施。

Academy 正在建设为一个公开的 open-source 教育生态系统。它包括实验室、课程体系、来源地图、数字出版、媒体基础设施、证书、学生作品集、导师路径、挑战项目以及类似 venture studio 的试点项目。

## 我们建设什么

- 区块链与 Bitcoin 教育；
- 用于网站、演示文稿、课程、证书、视频和研究笔记的 AI 工具；
- 数字图书馆和数字出版体系；
- 用于课程、直播、播客和网络研讨会的媒体中心；
- 机器人、传感器、数字健康、制造、纳米材料、计算和网络实验室；
- Academy Operating System，用于学习路径、作业、评审、证书、作品集、导师、职业路径和项目试点。

## 原则

ABC4RD Academy 通过开放来源、实践任务、正确署名、负责任的实验和公共学习基础设施来教授技术。
