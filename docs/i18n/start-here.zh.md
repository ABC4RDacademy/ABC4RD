# 从这里开始：ABC4RD Academy

Translation status: draft, requires native review.

ABC4RD Academy 是一个面向区块链和人工智能的国际教育计划，建设开放课程、研究实验室、
证书模板和演示项目，用于可验证的数字基础设施学习。

## 我们是谁

ABC4RD Academy 通过开放来源、实践任务、谨慎署名、负责任的实验和公共学习基础设施来
教授技术。

## 语言

英语是 GitHub 上的 canonical working language。公共入口正在准备阿拉伯语、中文、
英语、法语、俄语和西班牙语版本。

## 课程 01 摘要

课程 01 是 `Blockchain Academy: From Bitcoin to Verifiable Infrastructure`。
它从 Bitcoin 基础开始，谨慎介绍 smart contract 的思维模型，并连接到证书、溯源和信任
边界。

## 证书免责声明

`ABC4RD Blockchain Foundations` 证书是教育完成记录。它不是认证学位、许可证、
就业保证或金融资格。

## Bulletin #1 摘要

Research Bulletin #1 介绍第一组公开基础材料：首页、Course 01、测试证书、研究简报
和 ERC-20 localnet 学习演示。

## 开始

- 阅读 `docs/start-here.md`。
- 打开 Course 01。
- 运行或审阅 ERC-20 localnet demo。
- 审阅证书模板。
- 通过 GitHub Issues 或 Discussions 参与贡献。
