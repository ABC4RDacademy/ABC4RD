## Назначение

Это первая отраслевая версия рейтинга ABC4RD Academy по блокчейн-инструментам
для разработчиков. Она измеряет публичную техническую глубину, open-source
поверхность, качество документации, официальную событийную видимость, маршруты
для сотрудничества, прозрачность управления и текущий продуктовый импульс.
