这是 ABC4RD Academy 的公开每日运营仪表板。

它把 `24/7` 节奏转化为清晰且可重复的 daily package，而不是噪音。

## 这里固定的内容

- 每日 source 区块;
- 每日 curriculum 更新;
- issue draft digest 层的每日纪律;
- 在周五发布前完成 weekly ranking 准备。
