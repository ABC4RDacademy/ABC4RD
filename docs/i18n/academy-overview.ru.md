# ABC4RD Academy

Статус перевода: draft, requires native review.

ABC4RD Academy — международная исследовательская и образовательная инициатива по блокчейну, Bitcoin, искусственному интеллекту, открытым вычислениям, робототехнике, сенсорам, цифровой медицине, цифровому производству, наноматериалам, сетям и проверяемой цифровой инфраструктуре.

Академия строится как публичная open-source образовательная экосистема. В нее входят лаборатории, учебные программы, карты источников, цифровое издательство, медиа-инфраструктура, сертификаты, портфолио студентов, наставнические треки, челленджи и пилотные проекты в формате venture studio.

## Что мы создаем

- обучение по blockchain и Bitcoin;
- AI-инструменты для сайтов, презентаций, курсов, сертификатов, видео и исследовательских заметок;
- цифровую библиотеку и цифровое издательство;
- медиацентр для лекций, стримов, подкастов и вебинаров;
- лаборатории по робототехнике, сенсорам, цифровой медицине, производству, наноматериалам, вычислениям и сетям;
- Academy Operating System для учебных траекторий, заданий, проверки, сертификатов, портфолио, наставников, карьерных путей и пилотных проектов.

## Принцип

ABC4RD Academy обучает технологиям через открытые источники, практическую работу, корректную атрибуцию, ответственное экспериментирование и публичную образовательную инфраструктуру.
