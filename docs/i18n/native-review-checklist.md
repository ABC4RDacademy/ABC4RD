# Native Review Checklist

Status: source.

This checklist is for reviewing ABC4RD Academy public entry pages in Arabic,
Chinese, French, Russian, and Spanish against the canonical English source.

Canonical source:

`docs/i18n/start-here.en.md`

## Review Scope

Only review the public entry doors during Launch Sprint 02:

- What is ABC4RD Academy;
- Start here;
- Course 01 summary;
- Certificate disclaimer;
- Bulletin #1 or #2 summary.

Do not translate or rewrite the full repository set yet.

## Required Checks

- [ ] Brand spelling is preserved: `ABC4RD Academy`.
- [ ] GitHub namespace is preserved: `ABC4RDacademy`.
- [ ] Translation status is visible.
- [ ] Disclaimer is not softened.
- [ ] Certificate text does not imply accreditation.
- [ ] Course text does not imply investment advice.
- [ ] Demo text does not imply token value or mainnet launch.
- [ ] Partnership, sponsorship, endorsement, and affiliation claims are not
  added.
- [ ] Links remain intact.
- [ ] Technical terms are understandable to the target audience.

## Review Status Values

Use the existing translation policy:

- `draft`;
- `requires native review`;
- `reviewed`;
- `domain reviewed`;
- `deprecated`.

## Reviewer Note Template

```text
Language:
Reviewer:
Date:
File reviewed:
Status after review:
Issues found:
Suggested edits:
Ready for public use: yes/no
```

## Safety Rule

If the reviewer is unsure about a technical, legal, financial, medical,
security, or certification phrase, keep the page marked
`requires native review` and request domain review.
