# Start Here: ABC4RD Academy

Translation status: source.

ABC4RD Academy is an international blockchain and AI education initiative
building open courses, research labs, certificates, and demo projects for
verifiable digital infrastructure.

## Who We Are

ABC4RD Academy teaches technology through open sources, practical work, careful
attribution, responsible experimentation, and public learning infrastructure.

## Languages

English is the canonical GitHub working language. Public entry points are being
prepared in Arabic, Chinese, English, French, Russian, and Spanish.

## Course 01 Summary

Course 01 is `Blockchain Academy: From Bitcoin to Verifiable Infrastructure`.
It starts with Bitcoin foundations, introduces smart-contract mental models
carefully, and connects the work to credentials, provenance, and trust limits.

## Certificate Disclaimer

The `ABC4RD Blockchain Foundations` certificate is an educational completion
record. It is not an accredited degree, license, employment guarantee, or
financial qualification.

## Bulletin #1 Summary

Research Bulletin #1 introduces the first public baseline: homepage, Course 01,
test certificate, research bulletin, and ERC-20 localnet learning demo.

## Start

- Read `docs/start-here.md`.
- Open Course 01.
- Run or review the ERC-20 localnet demo.
- Review the certificate template.
- Contribute through GitHub Issues or Discussions.
