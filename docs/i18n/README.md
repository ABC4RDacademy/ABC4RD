# ABC4RD Academy Internationalization

ABC4RD Academy uses English as the primary working language for GitHub, with public entry points translated into the six official United Nations languages:

- English;
- Arabic;
- Chinese;
- French;
- Russian;
- Spanish.

## Translation Rule

Translations may be drafted with AI, but public-facing translated material should be marked as draft until reviewed by a fluent speaker or domain expert.

Do not translate legal, medical, financial, compliance, RWA, security, or safety-critical statements in a way that changes their meaning.
