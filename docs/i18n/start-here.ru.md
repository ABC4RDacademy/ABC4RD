# Начните здесь: ABC4RD Academy

Статус перевода: draft, requires native review.

ABC4RD Academy — международная образовательная инициатива по блокчейну и
искусственному интеллекту, которая создает открытые курсы, исследовательские
лаборатории, сертификаты и demo-проекты для проверяемой цифровой
инфраструктуры.

## Кто мы

ABC4RD Academy обучает технологиям через открытые источники, практическую
работу, корректную атрибуцию, ответственное экспериментирование и публичную
образовательную инфраструктуру.

## Языки

Английский язык является canonical working language для GitHub. Публичные
входные страницы готовятся на арабском, китайском, английском, французском,
русском и испанском языках.

## Курс 01

Курс 01: `Blockchain Academy: From Bitcoin to Verifiable Infrastructure`.
Он начинается с основ Bitcoin, аккуратно вводит ментальные модели
smart-contract систем и связывает обучение с сертификатами, provenance и
границами доверия.

## Disclaimer по сертификату

Сертификат `ABC4RD Blockchain Foundations` является образовательной записью о
прохождении. Это не аккредитованная степень, не лицензия, не гарантия
трудоустройства и не финансовая квалификация.

## Bulletin #1

Research Bulletin #1 представляет первую публичную базу: главную страницу,
Course 01, тестовый сертификат, исследовательский бюллетень и ERC-20 localnet
learning demo.

## Начать

- Прочитайте `docs/start-here.md`.
- Откройте Course 01.
- Запустите или проверьте ERC-20 localnet demo.
- Посмотрите шаблон сертификата.
- Внесите вклад через GitHub Issues или Discussions.
