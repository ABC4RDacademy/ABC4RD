# ABC4RD Academy Translation Policy

ABC4RD Academy is an international educational initiative. Multilingual access is important, but accuracy is more important than volume.

## Language Model

Primary GitHub working language:

- English.

Official public translation targets:

- Arabic;
- Chinese;
- French;
- Russian;
- Spanish.

Together with English, these are the six official languages of the United Nations.

## Public Publication Rule

Every public article-level ABC4RD material should exist in the six official
United Nations languages:

- English;
- Arabic;
- Chinese;
- French;
- Russian;
- Spanish.

This applies to:

- articles;
- essays;
- bulletins;
- weekly digests;
- research notes intended for public reading;
- release notes;
- milestone notes when they are used as public-facing updates.

If a material exists in only one language, it is not yet fully compliant with
the ABC4RD multilingual publication standard.

## Status Labels

Use these labels in translated documents:

- `source` - original or canonical source text;
- `draft` - AI-assisted or human draft;
- `requires native review` - needs review by fluent speaker;
- `reviewed` - reviewed for language quality;
- `domain reviewed` - reviewed by a subject-matter expert;
- `deprecated` - no longer current.

## Rules

- Do not translate private information.
- Do not remove disclaimers.
- Do not soften legal, medical, security, financial, or safety warnings.
- Do not imply partnership, certification, accreditation, or endorsement unless publicly verified.
- Keep source links and attribution intact.
- Keep thematic preview images aligned across language versions of the same public article.
- Translate image alt text and captions when the article is translated.
- If a preview image contains visible words, labels, slogans, or headings, that
  visible text must match the language of the material that uses it.
- If localized image variants do not yet exist, use a text-free preview image
  rather than a preview with mismatched language.
- If uncertain, mark the translation as `requires native review`.

## Review Checklist

Before marking a translation as reviewed:

1. brand spelling is correct: `ABC4RD Academy`;
2. GitHub namespace is correct: `ABC4RDacademy`;
3. no unsupported partnership claim is added;
4. technical terms are not distorted;
5. disclaimers are preserved;
6. links still work;
7. status and date are visible;
8. preview image is present when the source article has one;
9. preview alt text matches the translated topic.
