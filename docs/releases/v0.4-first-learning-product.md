## Purpose

This release turns ABC4RD Academy from an infrastructure map into a working educational system with a first usable course, student path, challenge, certificate preview, and public bulletin.

## Included Work

- Flagship course: `Blockchain Foundations: From Bitcoin to Verifiable Infrastructure`;
- 8 real module files;
- student submission folder;
- student submission template;
- student journey v1;
- 7-day Blockchain Foundations challenge;
- certificate verification preview;
- research bulletin #1;
- GitHub permission check;
- milestone and issue set for v0.4;
- labels for course, student journey, credentials, challenge, and release tracking.

## Public Student Flow

1. Student opens Module 01.
2. Student reads assignment.
3. Student copies `submission-template.md`.
4. Student creates a public submission file.
5. Student opens a pull request.
6. Maintainer reviews.
7. Student receives completion status.
8. Student links the artifact in a portfolio.
9. Certificate verification page can be used as a demo model.
