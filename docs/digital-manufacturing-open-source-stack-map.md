# ABC4RD Digital Manufacturing Lab Open-Source Stack Map

ABC4RD Digital Manufacturing Lab is the Academy block for CAD/CAM, digital thread, OPC UA, MTConnect, machine data, CNC literacy, additive manufacturing, simulation, and manufacturing provenance.

## First Scope

| Layer | Open-source stack | First pilot | Time | Cost |
|---|---|---|---:|---:|
| CAD | FreeCAD, OpenSCAD, Open Cascade | parametric part library | 1-5 days | $0 |
| Slicing/additive | Cura, PrusaSlicer | 3D print workflow lesson | 1-3 days | $0-50 plus printer |
| Machine data | MTConnect, OPC UA | simulated machine telemetry | 2-10 days | $10-150/month |
| OPC UA libraries | open62541, Eclipse Milo | machine integration API demo | 3-14 days | $0-200/month |
| CNC/control | LinuxCNC | CNC safety and control study | 1-4 weeks | hardware-dependent |
| Digital thread | GitHub + metadata + CAD files | source-to-part traceability demo | 2-10 days | $0-100/month |

Create a digital thread demo: CAD model, revision history, simulated machine data, source verification, manufacturing note, and QR-linked documentation page.

Production note: no real machine control or safety-critical automation without industrial safety review.
