# Visual Identity

ABC4RD Academy should look consistent without over-designing the research work.

## Naming

- Brand text: `ABC4RD Academy`
- GitHub namespace target in public links: `ABC4RDAcademy`
- Current GitHub display may appear as `ABC4RDacademy`; links still resolve.

## Tone

- Source-backed.
- Technical.
- International.
- Educational.
- No hype.
- No fake affiliation.

## Colors

| Use | Color |
| --- | --- |
| Deep ink | `#17202a` |
| Source blue | `#1d4ed8` |
| Research green | `#0f766e` |
| Caution orange | `#9a3412` |
| Light paper | `#f5f7fa` |

## Badges

Recommended badges:

- `evidence-source_backed`
- `license-review_required`
- `contributions-welcome`
- `status-active`
- `status-draft`

## Repository README Pattern

1. One-sentence purpose.
2. Who it is for.
3. Current status.
4. Main documents.
5. How to contribute.
6. Safety/disclaimer block.
