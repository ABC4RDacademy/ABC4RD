# Events And Conferences Map

This map tracks conferences, forums, workshops, seminars, and hackathons that
matter to ABC4RD Academy research and education.

Inclusion here does not imply attendance, sponsorship, partnership, or
endorsement.

## Event Review Fields

For each event, track:

- official link;
- organizer;
- region;
- topic fit;
- speakers and sponsors, if public;
- GitHub-connected projects;
- student opportunities;
- official media archive;
- outreach priority;
- verification status.

## Conference Output Types

ABC4RD Academy should produce:

- public source notes;
- speaker/resource maps;
- student reading paths;
- glossary improvements;
- "requires verification" lists;
- post-event digests;
- GitHub issues for source review;
- no partnership claims unless explicit.

## Outreach Rule

Before writing to an organizer or speaker:

1. Publish the note.
2. Cite the official source.
3. Add disclaimer.
4. Ask only for correction or preferred official links.
5. Do not ask for promotion.
