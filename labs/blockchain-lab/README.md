# ABC4RD Blockchain Lab

This lab is the ABC4RD Academy environment for blockchain education and controlled pilots: Bitcoin, Ethereum/EVM, Solidity, Foundry, Hardhat, OpenZeppelin, DAO simulations, RWA/tokenization models, appchains, L2, privacy, wallets, oracles, and security.

The first local preview is:

- `blockchain-lab-preview.html`

## First Pilot Recommendation

Start with:

1. Foundry + Anvil local chain.
2. OpenZeppelin Contracts.
3. Scaffold-ETH 2 frontend.
4. ERC-20 education token on localnet only.
5. Certificate NFT or non-transferable credential prototype on testnet.
6. Safe + Snapshot DAO classroom simulation.
7. RWA mock registry with a strong disclaimer: token does not equal legal ownership.

All first pilots must be localnet or testnet. No investment promises, no public sale, no mainnet deployment, and no legal ownership claim without legal and security review.
