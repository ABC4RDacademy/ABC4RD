# ABC4RD Messenger Stack Lab

This lab explores how ABC4RD Academy can launch its own messenger layer for students, mentors, cohorts, research tracks, announcements, and open-source contribution work.

The first local preview is:

- `messenger-preview.html`

It is not a production messenger. It is a clickable concept screen showing what the first academy messenger landing and chat experience could look like.

## Recommended First Pilots

1. Zulip for education-focused topic-based academy chat.
2. Matrix Synapse + Element for sovereign long-term messaging.
3. Mattermost or Rocket.Chat for staff operations.

## What This Lab Should Produce

- Open-source messenger stack map.
- Student download/onboarding flow.
- Room/channel model.
- Cost and deployment plan.
- Moderation and privacy policy.
- First production deployment checklist.
