# ABC4RD Revenue Operations Lab

This lab defines how ABC4RD Academy sells courses, accepts payments, receives donations, handles crypto donations, tracks revenue, manages refunds, and keeps financial operations separate from educational claims.

## MVP

Time: 1-3 days for policy and setup.

Cost: $0-100/month before payment processing fees.

## First Launch Order

1. choose legal/account path;
2. publish pricing and donation policy;
3. open waitlist;
4. create donation page;
5. create paid beta cohort checkout;
6. add crypto donations only after wallet/accounting policy is approved.
