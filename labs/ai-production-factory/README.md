# ABC4RD AI Production Factory

This is the top-level production layer for AI-assisted educational assets: websites, presentations, bulletins, certificates, courses, research assistants, designs, videos, translations, agent workflows, assessments, and templates.

- Stack map: `../../docs/ai-production-factory-open-source-stack-map.md`
- Preview: `ai-production-factory-preview.html`
- Watch results: `ai-production-watch-results.csv`

## First Pilot Recommendation

Start with four visible products:

1. AI course/lab landing page generator.
2. AI lecture presentation generator.
3. ABC4RD weekly research bulletin generator.
4. Certificate and badge mock verification page.

## Time and Budget

| Version | Time | Cost |

AI drafts. Humans approve. Sources, licenses, claims, and private data must be checked before publication.
