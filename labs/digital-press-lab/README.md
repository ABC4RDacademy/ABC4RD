# ABC4RD Digital Press Lab

This lab explores how ABC4RD Academy can build a boutique digital publishing workflow for essays, books, research notes, articles, reports, preprints, proceedings, and open educational resources.

The first local preview is:

- `digital-press-preview.html`

## First Pilot Recommendation

Start with:

1. Quarto for essays, reports, books, HTML/PDF output.
2. Pandoc as conversion backbone.
3. Zotero/BibTeX for citations.
4. GitHub for versioning and editorial review.
5. Typst/Paged.js for polished PDF and print-style outputs.
6. Later: Pressbooks, PubPub, OJS, OMP, OPS.
