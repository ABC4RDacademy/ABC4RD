# ABC4RD Challenge and Hackathon Lab

Practical challenge programs so students build, submit, review, and improve real work.

First stack: GitHub Issues, GitHub Classroom, Markdown task templates, PrairieLearn, nbgrader, Discord/Zulip/Discussions for event coordination.

MVP: 2-7 days. Cost: $0-100/month.

First challenges:

- 7-day Bitcoin challenge;
- Solidity testnet challenge;
- AI website challenge;
- robotics simulation challenge;
- FHIR synthetic patient challenge;
- sensor dashboard challenge;
- open-source first PR challenge.
