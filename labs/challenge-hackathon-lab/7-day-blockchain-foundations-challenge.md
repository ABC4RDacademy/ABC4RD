# 7-Day Blockchain Foundations Challenge

Status: v0.1 draft.

Time to launch MVP: 1-3 days.

Estimated operating cost: $0-100/month.

## Goal

This challenge helps students move from passive reading to public work.

By the end of 7 days, each student should have:

- one GitHub submission;
- one source-backed explanation;
- one small diagram/table/glossary entry;
- one reviewed pull request or issue;
- one portfolio-ready artifact;
- eligibility for a draft completion certificate.

## Rules

- Use public sources.
- Cite sources.
- Do not copy copyrighted material.
- Do not make financial, investment, legal, medical, or production deployment claims.
- Do not publish private data.
- Ask for review, not promotion.

## Day 1: What Is Trust?

Task:

- write 250 words explaining why people need ledgers, signatures, witnesses, audits, or blockchains.

Output:

- `day-01-trust.md`.

## Day 2: Bitcoin Basics

Task:

- define block, transaction, UTXO, input, output, fee, miner, node, wallet, and private key.

Output:

- glossary table.

## Day 3: Proof of Work vs Proof of Stake

Task:

- compare PoW and PoS using a simple table.

Output:

- comparison table with at least 3 sources.

## Day 4: Smart Contracts

Task:

- explain what a smart contract can and cannot do.

Output:

- short note using an ERC-20 example.

## Day 5: Tokens, Credentials, and RWA

Task:

- explain why a token is not automatically legal ownership.

Output:

- use-case evaluation using the ABC4RD framework.

## Day 6: DAO Governance

Task:

- draft a simple DAO proposal.

Output:

- proposal with purpose, decision rule, risks, and review checklist.

## Day 7: Final Submission

Task:

- create one GitHub pull request with the week’s best artifact.

Output:

- PR link;
- 3-5 sources;
- disclaimer;
- portfolio summary;
- next learning goal.

## Completion Criteria

A student completes the challenge when:

1. at least 5 of 7 daily tasks are completed;
2. one public GitHub submission is created;
3. sources are cited;
4. no unsafe claims are made;
5. a reviewer approves or requests minor revisions.

## Certificate

Eligible students may receive:

- `ABC4RD 7-Day Blockchain Foundations Challenge Certificate`.

Certificate status:

- draft educational certificate;
- not an academic degree;
- not professional licensure;
- not investment, legal, medical, or security certification.
