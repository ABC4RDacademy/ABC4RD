# ABC4RD Credentials and Certificates Lab

Certificates, Open Badges, verifiable credentials, QR verification pages, learner records, and course completion evidence.

First stack: W3C Verifiable Credentials, Open Badges, digitalbazaar/vc, CREDEBL, DID/VC verification pages.

MVP: 2-7 days. Cost: $0-100/month for a simple verification prototype. Serious credential infrastructure can cost $500-5000+ to build and harden.
