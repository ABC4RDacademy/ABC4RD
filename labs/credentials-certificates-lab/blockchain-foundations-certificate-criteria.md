# Blockchain Foundations Certificate Criteria

Status: v0.1 draft.

Certificate:

`ABC4RD Blockchain Foundations Completion Certificate`

## Minimum Requirements

A student is eligible for a draft completion certificate after:

1. completing at least 6 of 8 course modules;
2. completing the 7-Day Blockchain Foundations Challenge or equivalent assignment set;
3. opening at least one GitHub pull request;
4. receiving review approval;
5. adding one portfolio-ready artifact;
6. acknowledging the educational disclaimer.

## Certificate Fields

The verification page should include:

- student public name or handle;
- course name;
- certificate ID;
- issue date;
- completion status;
- reviewed modules;
- challenge status;
- reviewer/maintainer status;
- verification link or QR code;
- disclaimer.

## Certificate ID Format

Draft format:

```text
ABC4RD-BF-YYYY-NNNN
```

Example:

```text
ABC4RD-BF-2026-0001
```

## Verification Statuses

- `draft`;
- `educational demo verified`;
- `submitted for review`;
- `reviewed`;
- `revoked`;
- `expired`;
- `requires correction`.

## Disclaimer

This certificate is an educational record. It is not an accredited degree, professional license, investment credential, legal authorization, medical credential, security audit certificate, or production deployment approval.
