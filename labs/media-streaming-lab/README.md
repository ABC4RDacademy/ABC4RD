# ABC4RD Media Streaming Lab

This lab explores how ABC4RD Academy can build its own media and streaming product: video portal, live classes, webinars, podcast feeds, lecture archive, course embeds, transcripts, and controlled distribution to YouTube/Vimeo/social platforms.

The first local preview is:

- `media-hub-preview.html`

## First Pilot Recommendation

Start with:

1. MediaCMS for the Academy video/media portal.
2. Owncast for public live classes and launch streams.
3. OBS Studio for recording and live production.
4. FFmpeg and MediaInfo for conversion and media QA.
5. Video.js or HLS.js for embedded video pages.
6. Castopod for podcasts.
7. BigBlueButton later for classroom-style webinars.
