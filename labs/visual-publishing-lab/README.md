# ABC4RD Visual Publishing Lab

This lab explores how ABC4RD Academy can quickly produce websites, landing pages, presentations, bulletins, posters, certificates, and visual materials from repeatable open-source templates.

The first local preview is:

- `visual-factory-preview.html`

## First Pilot Recommendation

Start with:

1. Astro for public landing pages.
2. Docusaurus for curriculum/documentation sites.
3. Start Bootstrap / Flowbite / DaisyUI / shadcn/ui for ready blocks.
4. Slidev / Marp for presentations.
5. Typst for certificates and reports.
6. Coolify or Dokploy for deployment.
