# ABC4RD Digital Library Lab

This lab explores how ABC4RD Academy can build its own digital library for legal books, open access papers, official standards, datasets, lectures, reading paths, and student notes.

The first local preview is:

- `library-preview.html`

The current preview is not a production library. It is a clickable concept for reviewing the user journey, metadata fields, and legal source policy.

## First Pilot Recommendation

Start with:

1. Calibre-Web or Kavita for a readable ebook/PDF library.
2. Zotero for citation discipline.
3. BookStack or Wiki.js for library rules and reading paths.
4. Later: DSpace/InvenioRDM/Dataverse for serious institutional repository work.

## Non-Negotiable Rule

ABC4RD Library must store only legal content:

- public domain;
- open access;
- Creative Commons;
- original Academy material;
- explicit permission.

For copyrighted books without permission, store metadata and official links only.
