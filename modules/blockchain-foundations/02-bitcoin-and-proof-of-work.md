# Module 02: Bitcoin and Proof of Work

Course: Blockchain Foundations: From Bitcoin to Verifiable Infrastructure.

Status: v0.1 draft.

## Goal

Understand Bitcoin as a public monetary network using Proof of Work, UTXO, signatures, blocks, nodes, mining, and self-custody.

## Read

- Bitcoin whitepaper;

- Bitcoin Optech topic pages;
- one beginner-friendly explanation of UTXO.

## Learn

Bitcoin does not use account balances in the same way a bank account does. It uses unspent transaction outputs. A wallet helps create and sign transactions, but the network validates rules through nodes.
