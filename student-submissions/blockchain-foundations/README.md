# Blockchain Foundations Student Submissions

This folder stores public student submissions for:

`Blockchain Foundations: From Bitcoin to Verifiable Infrastructure`.

## How to Submit

1. Copy `submission-template.md`.
2. Rename it to your GitHub username:

```text
student-submissions/blockchain-foundations/[github-username].md
```

3. Complete the template.
4. Open a pull request.
5. Wait for review.

## Rules

- Use public sources.
- Cite sources.
- Do not copy copyrighted material.
- Do not include private data.
- Do not publish wallet keys, seed phrases, passwords, tokens, or private contact details.
- Do not make financial, investment, legal, medical, or production deployment claims.
